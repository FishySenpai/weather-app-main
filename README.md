## installation (windows)

### Windows (on powershell.exe)

```powershell
cd 
git clone https://github.com/ekinakkaya/weather-app.git
cd weather-app
npm run make
# the setup.exe file will be located at ./out/make/squirrel.windows/x64/
# run it and you're good
```
